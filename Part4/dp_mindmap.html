Dynamic Programming — Taxonomy Mind Map
Dynamic Programming (DP)
1. Prefix DP (1D)
DP on prefixes of a sequence without cross-element comparison.
  • Word Break House Robber Paint House Stock Trading (1-state) Climbing Stairs
2. Subsequence DP (Comparison-based Prefix DP)
DP where decisions come from using or skipping elements or comparing characters.
  • 2.1 1D Subsequence DP
    • LIS LDS MSIS (Max Sum Increasing Subsequence)
  • 2.2 2D Subsequence DP
    • LCS (Longest Common Subsequence) Edit Distance Subsequence Counting Pattern Matching DP
3. Interval / Cutting DP
DP on substrings or intervals using partition points.
  • Burst Balloons Matrix Chain Multiplication Stone Merging Integer Break Rod Cutting
4. Grid / Coordinate DP
DP based on movement across coordinates.
  • Unique Paths Min Path Sum Obstacles Grid Chessboard DP
5. Knapsack DP
DP based on pick/skip with capacity constraints.
  • 0/1 Knapsack Unbounded Knapsack Bounded Knapsack Combination vs Permutation Knapsack
6. Tree / Graph DP
  • Tree DP (Subtree DP) Tree Diameter Vertex Cover on Tree DAG DP
7. Bitmask DP
  • TSP Assignment Problem Subset DP
8. Digit DP
  • Count ≤ N Digit Sum Constraints No Repeated Digits
9. Counting DP
  • Ways Counting Pattern Counting Lexicographic Reconstruction DP
10. State-Machine / Game DP
  • Multi-State Stock DP Nim / Grundy Games Finite-State Automaton DP
